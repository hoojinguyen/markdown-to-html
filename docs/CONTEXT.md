# Markdown-to-HTML Converter Context

A robust, high-quality, pure-Bash Markdown parsing and HTML rendering tool.

## Language

**Markdown Document**:
The input text document containing structured Markdown elements to be converted.
_Avoid_: md file, input string

**HTML Render**:
The semantic HTML output generated by the converter, containing structured elements and optional CSS visual themes.
_Avoid_: output page, html file

**Visual Theme**:
A named styling configuration that defines the typography, colors, padding, and general aesthetic of the HTML Render.
_Avoid_: design, theme config, stylesheet

**Theme Loader**:
The module responsible for parsing visual theme variables and outputting the HTML standalone header and footer templates.
_Avoid_: themes.sh, styling loader

**Link Reference Registry**:
A collection of custom, name-based URL and title mappings parsed in a first-pass to support reference-style link and image generation.
_Avoid_: env ref, variable mapper

**Inline Transform Engine**:
The formatting utility that translates inline styling syntax (bold, italic, inline code, images, links) into corresponding inline HTML elements.
_Avoid_: inline_transform, sed replacer

**Block Parser**:
The line-based state machine that processes document structures (headers, paragraphs, lists, tables, blockquotes, and code blocks).
_Avoid_: parser loop, state manager
